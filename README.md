# Лендинг-портфолио — Соколова Анастасия

Одностраничный лендинг-портфолио для продуктового и UX/UI дизайнера Соколовой Анастасии. Проект собран на «голом» HTML/CSS/JS без фреймворков и ориентирован на понятную архитектуру и лёгкое масштабирование портфолио.

## Структура

- `index.html` — основная страница.
- `styles/variables.css` — дизайн‑токены (цвета, типографика, отступы).
- `styles/base.css` — базовые стили, кнопки, формы, карточки.
- `styles/layout.css` — сетка, секции, адаптив.
- `styles/components.css` — стили компонентов (карточки, FAQ, слайдер).
- `scripts/portfolio-data.js` — данные с проектами (массив `projects`).
- `scripts/main.js` — логика: плавный скролл, аккордеон FAQ, рендер портфолио, слайдер отзывов, форма.

## Запуск

1. Откройте `index.html` в браузере (двойной клик по файлу или через локальный http‑сервер).
2. Для корректной работы ES‑модулей (подключение `portfolio-data.js` в `main.js`) рекомендуется запускать через простой статический сервер:

```bash
npm install -g serve
serve .
```

## Как добавлять новые проекты

1. Откройте `scripts/portfolio-data.js`.
2. Добавьте новый объект в массив `projects` с полями:
   - `id`, `title`, `category`, `year`, `result`, `shortDescription`, `tags`, `coverImage`.
3. Разметка в `index.html` обновится автоматически через `initPortfolioGrid(projects)` без правок вёрстки.
